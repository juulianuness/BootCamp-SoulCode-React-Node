Variaveis

Um container p um valor, como numeros que podemos usar em uma operação de adição ou uma sequencia de texto.
Seu conteudo pode mudar ao decorrer do codigo.
Codigo sem variavel não pode existir, pois elas são a base da pagina, p salvar dados ou intermediarios de algum processamento.
Serve como memoria temporária de daados, seja, variaveis são informações temporárias, se atualizar a pagina, as informações são perdidas.
Variaveis NÃO são o valor em si, são apenas os locais em que está sendo guardado esse valor

Como declarar uma variavel

let + valor (instrução) + ;
ex: let nome
let idade

Regras Basicas

Usar nomes descritivos para variaveis, que justifique o que ela faz.
COMEÇAR COM LETRA, SUBLINHADO OU $
Não usar palavras reservadas (let, var, function, if...)
camelCase, começa minusculo e as proximas palavras começam com letra maiuscula

Para defnir um valor a variavel

Let nome, let idade, let nota

let nome = 'Julia';
let idade = 29;
let nota = 9.8;
(as variaveis já tem um valor, mas pode mudar depois)
let nome = "Julia N" ( o valor antigo é perdido e substituido pelo novo valor, a menos que eu code para guardar)

console.log(nome, idade) -> Mostra lado a lado
console.log(nome) -> Mostra 1 em cada linha
console.log(idade) -> Mostra 1 em cada linha
console.log(idade, "anos") -> Mostra a idade com uma mensagem após

Const

Impede de alterar o valor da variavel (no lugar do let) e com letras maiusculas. (const não muda valor, let muda)

JS é uma linguagem de programação de tipagem dinamica (texto, vetor, objeto, numero) e pode atribuir um valor a uma variavel sem declarar o tipo da variavel, o tipo vai ser declarado pela entrada.

let hoje = "08/07/2025"
hoje = 8
hoje = true;
(mas é sempre melhor informar o tipo p não dar conflito)

Operadores Aritimeticos

let numero1 = 6, numero2 = 3;
console.log(numero1, numero2); // 6 3
console.log(numero1 + numero2); // 9
console.log(numero1 - numero2); // 3
console.log(numero1 * numero2); // 18
console.log(numero1 / numero2); // 2
console.log(numero1 ** numero2); // 216
console.log(numero1 % numero2); // 0

oOperadores de atribuição

let numero3 = 3;
numero3 += 4; // 7
numero3 -= 2; // 5
numero3 *= 4; // 20
numero3 /= 10; // 2
console.log(numero3); // 2

Operadores de Incremento

let numero4 = 8;
console.log(numero4); // 8
console.log(++numero4); // 9
console.log(numero4++); // 9
console.log(numero4--); // 10
console.log(--numero4); // 8

numero4++;
console.log(numero4); // 9
