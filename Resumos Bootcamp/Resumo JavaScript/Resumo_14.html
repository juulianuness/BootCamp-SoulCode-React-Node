Funções

Bloco de codigo reutilizavel que executa uma tarefa específica

Entrada + processamento + saida


ex: Função somar (n1 + n2) {
return n1 + n2;
}

n1 e n2 dos parenteses são os parametros, "return" quando eu quero o resultado da operação (mas não estou pedindo para mostrar na tela, apenas estou informando o que quero) e esse valor que eu pedi é o argumento da minha função

Só definir a função não quer dize que ela vai aparecer, preciso chamar ela fora do {}

a função só executa quendo coloco o ( )

Função Anonima

A variavel pode ser usada como função, desde que eu guarde ela em uma finção, precisa ser guardada dentro de uma variavel

const somar = function (n1, n2) {
let soma = n1 + n2;
return soma;
}
console.log(somar(12, 8)); // 20

// Funções de seta (Arrow Functions)

const subtrair = (n1, n2) => n1 - n2; let resultado = subtrair(15, 9); console.log(resultado); // 6 const quadrado = n => n**2; console.log(quadrado(8)); // 64

// Função de callback

setTimeout(() => { console.log("Acabou o tempo."); }, 3000); setInterval(() => { console.log("Passou 4 segundos."); }, 4000); console.log("Final do código.");
