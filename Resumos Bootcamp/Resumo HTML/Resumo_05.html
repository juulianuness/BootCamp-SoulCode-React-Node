RESUMO PESSOAL BOOTCAMP DIA 05

Formulario

Coletor de informação que permite que insiram dados
< form

method "post" e "get"

forma que os dados serão processados
• post manda para algum lugar/servidor
• get, manda para o mesmo url que está

input type = entrada de informação

text, email, number, password, radio, checkbox, select (option),

placeholder

- mensagem padrão dentro da caixa de mensagem

atributo nome

- obrigatorio p referenciar o que está fora da info quando chega no backend

textarea

- campo de texto multilinha, normalmente deixado vazio para que o usuario coloque informação

button

Button form = formulario > enviar < button >

( para deixar o botão fora do formulario e ainda assim ele funcionar como submit tem que colocar um ID no formulario ) br no BUTTON a opção automatica já é o submit, que envia a informação p algum lugar

button type reset

- normalmente não usado pois se a pessoa clicar errado ela perde tudo que digitou

button type button

- não envia dada, mas aciona oo JavaScript para a interação que ele solicitar por la

Grupo de opções

possuem valores pre definidos (eu coloco as opções)

input type radio

caixa de multipla escolha.
QAo usar o radio é necessario identificar o valor da marcação com VALUE e o ID precisa ser o mesmo em todas as opções para funcionar de escolher uma entre elas

checkbox

caixa de seleção que permite mais de uma escolha (até todas)

Label

Rotulo, ao clicar nele, o input em questão fica "ressaltado", ele meio que clica na caixa mesmo se o meu mouse tiver clicado no texto dessa mesma caixa.
PRECISA TER O MESMO NOME DO ID DO INPUT

Required

indica campos obrigatorios (não é uma tag, é só incluir a palavra no codigo)

min e max

Valores minimos e maximos de valor numerico num campo numerico

minlength e maxlength

quantidade minima e maxima de letras

Checked

Deixa a opção previamente marcada para o usuario

pattern

Padrão de validação usando expressões regulares(forma de determinar um padrão, tipo CPF)

selected

Seleciona previamente uma das opções do select

Disabled

Pode desabilitar o preenchimento de algum dos campos ou a sessão inteira

optgroup

Seleciona um grupo de opções dentro do grupo do select

auto complete: off

Desabilita a sugestão de preenchimento do usuario
