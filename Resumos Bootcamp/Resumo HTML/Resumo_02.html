HiperTexto

Texto inteligente onde consigo clicar nas palavras e ser direcionada ou interagir com ele

Marcação

Usam teg para estruturar e formatar seu conteúdo e possuem regras: abrindo e fechando colchetes angular: + que _ que (< >)

É possivel apenas colocar o nome da tag, sem ter que digitar os colchetes angular
e o proprio vs code coloca automaticamente, através do mecanismo EMMET ABREVIATION que fornece o auto complete

                              ex: p*4 (gera 4 paragrafos. pois pedi o P 4 vezes) 
                              ex: p>a ( atalho para a tag a href)

Diferença entre as tags B e Strong:

o B para acessibilidade não faz diferença semantica, apenas para a visualização.
o STRONG faz diferença na leitura/audio da acessibilidade, pois é como um "ponto de obs"

Algumas tags não precisam de fechamento: 

                              ex: < b > 
                              ex: < IMG >

Para conseguir visualizar na pagina algo que no VS CODE é uma TAG, é necessario usar a estrutura & lt; & gt; (porem, sem o espaço entre o & e as letras L e G)

Metadados

São colocados no HEAD e nõ aparecem no site.
Otimização para mecanismos de busca para melhorar a visibilidade da busca utilizando Author, Description e Keywords 

                              ex:<" meta name= "Author" content="Jaqueline" " 
                              ex:<" meta name= "Description" content="Psicologa" " 
                              ex:<" meta name= "Keywords" content="terapia, saude mental" "

a TAG pre vai respeitar os espaços estabelecidos no HTML
