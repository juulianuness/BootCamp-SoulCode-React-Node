WORD WIDE WEB - WWW

Sites possuem um endereço chamado URL que garantem acesso imediato a um site específico.
Chamamos de web o sistema que os sites estão organizados, as paginas que formam um site.

• Cliente side - paginas que eu acesso
• Server side - dados contidos no servidor que ele retorna quando eu solicito no acesso.
...... O servidor retorna 3 arquivos: HTML/CSS/JAVASCRIPT

WEB SITE

Coleção de paginas que estão relacionadas entre si, São bem rganizadas e acessiveis por meio da internet.
• Coleção de arquivos que fazem sentido juntos.

• HTML (esqueleto) = Determina cada coisa e onde cada coisa vai ficar.
• CSS (roupa/estilo) = Estilo, fonte, cor, alinhamento etc.
• JAVASCRIPT (cerebro) = Criar a logica no site, as interações.
