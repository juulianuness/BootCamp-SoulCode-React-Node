    • Sessão com links de navegação

    • Define todo conteudo principal do body/ tudo entre o header e o footer
  • Section
    • Sessão generica, Ajuda na estruturação de documentos longos: Quando você tem textos com vários blocos temáticos.
  • Article
    • Artigo usado em varios momentos (comentarios ou elementos)que são reutilizaveis no
  • ID
    • Identidficar (valor com palavras sem espaços, pode usar _ ou -)
  • Div
    • Define uma divisão/sessão de um documento/conteiner.
      uma Div não tem semantica, as demais tem. pode ser usada numa caixa de dialogo/alerta/aviso/popup
  • Spam
    • Tag de texto que não necessariamente é um paragrafo

    • A serve para colocar link, Href é a referencia dele, Target é para abrir em outra aba, Title é para uma legenda
