CSS

Linguagem de estilo Usada para aplicar visual a pagina

Tipos de Estilzação

Css Inline - Na mesma linha do HTML

Estilização Interna - Feito no HTML pela abertira e fechamento da tag style. no head

Estilização Externa - Criando um novo arquivo .CSS e linkando ele no HTML pela tag link

Estrutura

Propriedade : valor ;

Ex: color : red ;

Sintax e Seletores

Seletores São um mecanismo que nos permite selecionar grupos de elementos existentes da pagina de diversas formas

Exemplo

Propriedade1 : valor1;
Propriedade2 : valor2;
Propriedade3 : valor3;

Quando quero identificar um elemento ID, preciso colocar o # antes do nome dele no CSS
Quando quero identificar um elemento CLASS, preciso colocar o . antes do nome dele no CSS

Seletor Class

O nome da class posso repetir varias vezes, JA O ID NÃO POSSO REPETIR

# slogan .button { ... } - Dessa forma me refiro ao botão que está dentro da classe slogan

# slogan .button , #footer .button { ... } - Dessa forma me refiro aos botões que está dentro da classe slogan e footer

.carrossel h2 , carrossel h3, h4 { ... } - Dessa forma me refiro a todo h2 dentro de carrossel, a todo h3 dentro do carrossel e de todo h4 do arquivo html inteiro

Pseudo-classe

link:visited - quando um link for visitado
link:active - quando segurar o mouse em cima do link
link:hover - quando o mouse passar pelo link
link:focus - destaca parte da mensagem

Especificidade e herança

Especificidade Mecanismo que o CSS usa para determinar qual estilo será aplicado a um elemento quando existem regras de estilo conflitante
Cada seletor possui um valor de especificidade atribuido, que e calculado com base nos diferentes componentes do seletor
Quanto mis específico o seletor for, maior será a sua especificidade e maior será a probabilidade de que seu estio seja aplicado

Ordem de especificidade

P, DIV... ESPECIFICIDADE MODERADA

CLASSES, ATRIBUTOS, PSEUDO-CLASSES... ESPECIFICIDADE MEDIA

ID... ESPECIFICIDADE ALTA

P, DIV... ESPECIFICIDADE MODERADA

!IMPORTANT; ESPECIFICIDADE PRIORITÁRIA

                     (1,2,3)

1(id) , 2(classe, atributo, pseudo-classe) 3(tag)

ex: h2#titulo.titulo
(1,1,1)

Herança

Cor, fonte, tamanho de texto herdados do PAI, (nem todas as propriedades são herdadas) Body é um pai
