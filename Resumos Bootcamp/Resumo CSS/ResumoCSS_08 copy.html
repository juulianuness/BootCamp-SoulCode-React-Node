Flex Box

Centraliza um bloco de conteudo verticalmente
Faz com que os filhos de um container ocupe uma quantidade igual de altura e largura disponivel independente da alt. e lar. disponivel 

               Só funciona se eu habilitar o display-flex

O mesmo elemento pode ser tanto ITEM quanto CONTAINER para outros itens

Fazer todas as colunas de um layout commultiplas colunas adotem a mesma altura, mesmo que contenham uma quantidade diferente de conteudo

Row Indica que os flex itens irão se organizar em linha distribuindo-se horizontalmente na direção, o padrão é lado a lado

Column Indica que os flex itens irão se organizar em coluna distribuindo-se verticalmente, itens um a baixo do outro

Cada config. do flexbox altera a forma como algumas propriedades irão funcionar

Se eu to em linha, o eixo principal tambem é linha

Se eu to em coluna, o eixo principal tambem é coluna

Colocando o display-flex a tag vira container e o que tem dentro, vira Item

Se eu colocar start no justify-content, comea da linha.
Flex direction determina quem vai ser o eixo principal e vai justificar os itens nas respectivas linhas, conforme eu solicitar, se for em linha
