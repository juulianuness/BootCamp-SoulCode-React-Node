Atalhos de todas as aulas

  • Ctrl + alt + A Envolve a seleção em uma tag

  • Ctrl + b Esconde/volta a barra lateral esquerda

  • Ctrl + enter Cria uma proxima linha

  • Ctrl + k + s Salva todas as paginas abertas

  • Ctrl + c + Ctrl + v duplica a linha mesmo sem selecionar ela

  • Ctrl + D Edita as informações

  • Ctrl + Espaço Abre o emmet abreviation (caixa de opções)

  • Alt + Shift + seta duplica a linha

  • Alt + setas Move o texto de linha

  • Shift + tab Volta uma casa na indentação

  • Tab Indentação (4 em 4 casas)
